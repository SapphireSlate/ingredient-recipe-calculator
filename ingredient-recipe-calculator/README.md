## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone <repository-url>
```

2. Install dependencies
```bash
cd ingredient-recipe-calculator
npm install
```

3. Start the development server
```bash
npm start
```
